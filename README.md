Note: Make sure madlibs.txt is in the same folder as madlibs.py

How to make your own Madlibs story: Write out your story with your blank substitutions with a "$" on either side like so - $noun$
Make sure the story starts with a "#" and a number and ends with another "#"
To chose between more than 2 stories, madlibs.py will have to be modified accordingly
